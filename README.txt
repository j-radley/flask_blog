RUNNING THE WEBSITE FROM THE COMMAND LINE (Windows):
1) Set up virtual environment:
        py -m venv venv
        venv\Scripts\activate

2) Install dependencies:
        pip install -r requirements.txt


3) Run flask blog:
        flask run
